# Precoro Vacation Mode UI

Front-end for configuring “Vacation Mode” inside Precoro. Built with Vue 3, TypeScript, Vite, and Tailwind CSS.

## Features

- Date range selection for vacation periods.
- Substitute user pickers with async search.
- Backup approver management grouped by documents and steps.
- Toast notifications for update status.

## Stack

- Vue 3 (`<script setup>`) + TypeScript
- Vite
- Tailwind CSS
- Custom SVG icon set

## Scripts

- `npm run dev` – start local dev server
- `npm run build` – production build (vue-tsc + Vite)
- `npm run preview` – preview built app

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` to manage your Precoro vacation settings.
